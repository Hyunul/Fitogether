# FitTogether Web

React 기반의 FitTogether 웹 애플리케이션

## 기술 스택

- React
- TypeScript
- Vite
- TailwindCSS
- Firebase Auth

## 시작하기

### 필수 조건

- Node.js 18.x 이상
- npm 또는 yarn

### 설치

```bash
npm install
# 또는
yarn install
```

### 개발 서버 실행

```bash
npm run dev
# 또는
yarn dev
```

### 빌드

```bash
npm run build
# 또는
yarn build
```

## 환경 변수

`.env` 파일을 생성하고 다음 변수들을 설정하세요:

```
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```

## 주요 기능

- 소셜 로그인
- 운동 루틴 관리
- 챌린지 참여
- 인증 피드
- 리포트 확인
